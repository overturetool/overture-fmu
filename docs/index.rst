############
Overture FMU
############

Overture FMU exports VDM-RT models as FMUs. The export is carried out via C code or by wrapping the interpreter.

| Overture FMU has various dependencies to different projects as presented in :ref:`tool_dependencies`. For this reason, there are currently the following constraints:
| - Overture FMU works for Overture 3.6.4 for both the C-code generation approach and the tool-wrapping approach.

Please see :ref:`getting_started` for information on how to use Overture FMU for exporting FMUs from VDM-RT code.
